# Classifier

Classifier is a web app for organizing media folders on a NAS. It targets Docker-based deployment and focuses on classification, batch operations, thumbnail generation, and concurrent folder processing.

## Features

- Automatic classification for photo sets, videos, mixed folders, and comics
- Batch renaming with custom rules and preset scripts
- Fast ZIP compression for image directories
- Emby-compatible video thumbnail generation
- File moving into target directories
- Concurrent processing for multiple folders
- NAS-friendly deployment with Docker
